## 1. 背景・目的

### 1.1 背景
ベトナムでは屋台やローカル飲食店が日常生活や観光において重要な役割を果たしている。しかし、**原材料表示・アレルギー情報・英語対応が著しく不足**しており、日本でも個人経営の飲食店で同様の問題が存在する。

これにより、食物アレルギー保有者、ベジタリアン、宗教的食事制限を持つ人々が安心して食事をできないという社会課題が生じている。

### 1.2 目的
- 屋台・ローカル飲食店の**使用食材情報**をユーザーコミュニティで共有する
- 食物アレルギーや食事制限を持つ人が**事前に安全に店舗・メニューを選択**できる環境を提供する
- 日本とベトナムの飲食店における**食の情報格差**を埋める

## 2. ターゲットユーザー

| 区分         | 想定ユーザー                    | 主なニーズ |
|--------------|--------------------------------|-----------|
| メイン利用者 | 食物アレルギー保有者           | 危険食材を事前に把握したい |
| メイン利用者 | ベジタリアン／ビーガン         | 対応可能な店舗を知りたい |
| メイン利用者 | ムスリム（ハラール対応希望者） | ハラール対応情報を知りたい |
| 利用者       | ベトナム旅行者・在住外国人     | 言語の壁を越えて安全に食事したい |
| 利用者       | 現地住民                       | アレルギー情報や発見を共有したい |

## 3. 機能要件

### 3.1 機能一覧

| ID   | 機能名           | 概要                                                                 | 認証   |
|------|------------------|----------------------------------------------------------------------|--------|
| F-01 | ユーザー登録     | メールアドレスまたはユーザー名で登録                                 | 不要   |
| F-02 | ログイン/ログアウト | Cookie認証で状態を保持                                               | - / 要 |
| F-03 | 店舗一覧表示     | カード形式で表示（店名・ジャンル・対応タグ・写真）                   | 不要   |
| F-04 | 店舗詳細表示     | 使用食材一覧・アレルギー警告・メニュー例・レビュー・ユーザー投稿情報 | 不要   |
| F-05 | 検索・フィルタ   | キーワード検索＋条件検索（卵なし、ナッツなし、ハラール、ベジタリアン等） | 不要   |
| F-06 | 食材情報投稿     | 店舗の使用食材情報や補足情報を投稿                                   | 要     |
| F-07 | レビュー投稿     | 店舗に対する評価と感想を投稿                                         | 要     |
| F-08 | マイページ       | 自分が投稿した情報の一覧（簡易版）                                   | 要     |

### 3.2 権限ルール
- **閲覧機能**（店舗一覧・詳細・検索）：ログイン不要
- **投稿機能**（食材情報・レビュー）：ログイン必須
- **編集・削除**：原則として投稿者本人のみ可能

### 3.3 スコープ外（v0.1では実装しない）
- 地図表示（Google Maps APIなど）
- 本格的な多言語対応（ベトナム語）
- 店舗オーナー向け管理画面
- 投稿内容の管理者承認ワークフロー
- 画像アップロード機能（店舗写真）
- パスワードリセット機能

## 5. 用語定義

- **食事制限タグ**：卵、乳製品、ナッツ類、甲殻類、ハラール、ベジタリアン、ビーガンなど
- **アレルギー警告**：危険な食材が含まれる場合に目立つ表示を行う
- **ユーザー投稿**：一般ユーザーが自由に情報を追加・共有できる機能
